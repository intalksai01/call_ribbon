================================================================================
  PART 2: WHAT YOU NEED TO DO (One-Time Setup)
================================================================================

STEP 1: BUILD THE WIDGET
-------------------------
cd widget
npm install
npm run build

Output: dist/ribbon.js and dist/ribbon.css


STEP 2: DEPLOY WIDGET TO CDN
-----------------------------
Option A: Netlify (Easiest)
   cd dist
   netlify deploy --prod
   
Option B: AWS S3 + CloudFront
   aws s3 sync dist/ s3://your-bucket/ribbon/v1/
   
Option C: Vercel
   vercel --prod

Result: Your CDN URLs
   https://cdn.yourcompany.com/ribbon.js
   https://cdn.yourcompany.com/ribbon.css


STEP 3: DEPLOY API SERVER
--------------------------
Option A: Heroku
   cd api
   heroku create your-ribbon-api
   git push heroku main
   
Option B: Your own server
   cd api
   npm install
   node server.js

Result: Your API URL
   https://api.yourcompany.com


STEP 4: CREATE CLIENT API KEYS
-------------------------------
Edit api/server.js and add client:

const clients = {
  'client-abc-123': {
    name: 'Client Name',
    exotelToken: 'their_exotel_token',
    exotelUserId: 'their_user_id',
    allowedDomains: ['client-domain.com'],
    monthlyCallLimit: 10000,
    plan: 'enterprise'
  }
};


STEP 5: GIVE CLIENT INTEGRATION INFO
------------------------------------
Provide each client:
  1. API Key: client-abc-123
  2. CDN URLs (from Step 2)
  3. docs/CLIENT_GUIDE.md

================================================================================
  PART 3: WHAT YOUR CLIENTS NEED TO DO (Simple Integration)
================================================================================

CLIENT INTEGRATION (3 Lines of Code)
------------------------------------

1. Add to <head>:
   <link rel="stylesheet" href="https://cdn.yourcompany.com/ribbon.css">

2. Add before </body>:
   <script src="https://cdn.yourcompany.com/ribbon.js"></script>

3. Initialize:
   <script>
     ExotelCallRibbon.init({
       apiKey: 'client-abc-123',  // You provide this
       position: 'bottom'
     });
   </script>

DONE! The ribbon appears on their page.


OPTIONAL: Set Customer Data
----------------------------
When they select a customer:

ExotelCallRibbon.setCustomer({
  phoneNumber: '+919876543210',
  name: 'John Doe',
  customerId: '12345'
});
