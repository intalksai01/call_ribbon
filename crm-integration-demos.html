🔧 Integration Instructions

1

Include the Widget Script

Add the IntalksAI Call Ribbon widget to your CRM page:

<script src="https://your-domain.com/intalksai-call-ribbon.js"></script>
2

Initialize the Ribbon

Configure the widget with your API credentials and callbacks:

ExotelCallRibbon.init({ apiKey: 'your-api-key', apiUrl: 'https://your-api-domain.com', position: 'bottom', // or 'floating' onCallEvent: function(event, data) { // Handle call events in your CRM console.log('Call Event:', event, data); } });
3

Load Customer Data

When a user selects a customer in your CRM, update the ribbon:

// When customer is selected in CRM ExotelCallRibbon.setCustomer({ phoneNumber: '+1-555-123-4567', name: 'Sarah Johnson', email: 'sarah@company.com', customerId: 'CRM-12345' });
4

Customize Styling

Override CSS to match your CRM's design system:

.intalksai-call-ribbon { --primary-color: #your-brand-color; --font-family: 'Your CRM Font'; --border-radius: 8px; }
